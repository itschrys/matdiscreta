# Guia de estudos de matematica discreta com NotebookLM
## Sobre o Projeto
Este repositório foi desenvolvido como um caderno temático de estudos sobre Matemática Discreta utilizando o NotebookLM como ferramenta de apoio.

O objetivo principal foi aprofundar conhecimentos em:
- Lógica Proposicional
- Teoria dos Conjuntos
- Relações e Funções
- Grafos
- Indução Matemática
- Combinatória

Além disso, o projeto busca explorar técnicas de engenharia de prompt para melhorar a qualidade das respostas geradas pela IA.

# Objetivos de Estudo

- Consolidar conceitos fundamentais de Matemática Discreta;
- Criar resumos estruturados para revisão;
- Desenvolver prompts reutilizáveis para estudo;
- Aprender a utilizar IA como ferramenta de apoio acadêmico;
- Documentar dificuldades e melhorias durante o processo.

---

## Materiais utilizados no NotebookLM

- Slides disponibilizados pelo professor da disciplina;
- Videoaulas complementares publicadas pelo docente;
- Anotações pessoais desenvolvidas durante os estudos;

# Engenharia de Prompts

## Prompt 1
"Explique lógica proposicional de forma simples e com exemplos do cotidiano."

### Resultado
A IA respondeu com exemplos básicos de tomadas de decisão e operadores lógicos.

### Problema encontrado
A resposta ficou superficial.

### Ajuste realizado
"Explique lógica proposicional com exemplos computacionais e tabelas verdade."

### Resultado após melhoria
A resposta ficou mais técnica e adequada para revisão acadêmica.

---

## Prompt 2
"Gere exercícios de indução matemática com dificuldade crescente."

### Resultado
Foram gerados exercícios básicos e intermediários.

### Dificuldade
Algumas respostas não mostravam o passo a passo completo.

### Correção
"Resolva detalhadamente mostrando cada etapa da indução."

# Resumos Estruturados

## Lógica Proposicional
- Proposição: sentença verdadeira ou falsa;
- Conectivos lógicos:
  - AND (∧)
  - OR (∨)
  - NOT (¬)
  - IMPLICAÇÃO (→)

## Teoria dos Conjuntos
- União
- Interseção
- Diferença
- Complemento

## Grafos
- Vértices
- Arestas
- Grafos direcionados
- Caminhos mínimos

---

# Glossário

| Conceito | Definição |
|---|---|
| Proposição | Frase com valor lógico |
| Grafo | Estrutura formada por vértices e arestas |
| Indução Matemática | Técnica de prova matemática |
| Relação | Associação entre elementos |

# Prompts Reutilizáveis

## Para Revisão
"Resuma o conteúdo de [tema] em tópicos objetivos."

## Para Exercícios
"Gere 5 exercícios sobre [tema] com gabarito."

## Para Explicações
"Explique [tema] como se estivesse ensinando um estudante iniciante."

## Para Revisão Avançada
"Crie perguntas de prova sobre [tema] com nível universitário."
